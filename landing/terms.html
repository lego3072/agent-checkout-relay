Service Scope

Agent Checkout Relay provides API endpoints and workflow tooling for routing buyer intent to Stripe checkout and related onboarding operations.

Billing

  • Starter is a monthly recurring subscription billed through Stripe.
  • DFY is a one-time implementation charge through Stripe.
  • Subscription renewals, payment method updates, and cancellations are managed in Stripe.

Access

  • Protected API endpoints require a valid API key tied to an active account.
  • Access may be suspended for failed payment, abuse, or policy violations.

Acceptable Use

You may not abuse, scrape, overload, reverse engineer, or attempt unauthorized access to the service.
